# Weekly Updates

This folder will track newly found UAV-CVGL papers. The intended workflow is:

1. Daily search finds candidate papers.
2. Candidate papers are parsed into structured summaries.
3. A weekly digest is generated.
4. Human verification decides whether the paper enters the curated paper list or leaderboard.

Status labels:

- `newly found`
- `parsed`
- `verified`
- `leaderboard added`
- `needs review`
