# Contribution Guide

Contributions are welcome. Please follow these rules before adding a paper, dataset, or leaderboard result.

## Add a paper

Please provide:

- Title
- Year and venue, if available
- Paper link
- Code link, if available
- Main category
- Tags
- Datasets and metrics used
- A short one-sentence core idea

## Add a dataset

Please provide:

- Dataset name
- Task type
- Public availability
- Download or request link
- Real or synthetic
- Supported metrics
- Whether it has dense satellite maps, local maps, heading annotations, or trajectory annotations

## Add a leaderboard result

Leaderboard entries must be reproducible and protocol-specific. Do not mix different splits, re-ranking settings, or modified protocols in the same table.
