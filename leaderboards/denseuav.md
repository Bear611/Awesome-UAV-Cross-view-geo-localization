# DenseUAV

DenseUAV is a cross-view geo-localization dataset for vision-based UAV self-positioning in low-altitude urban environments. The benchmark evaluates the ability to match drone-captured images against geo-tagged satellite tiles. This leaderboard tracks the Drone→Satellite retrieval task using the official train/test split.

Only final flagship method results on the official dataset protocol are listed. Automatically reviewed rows are marked `verified=false` until manual checking.
