# Game4Loc

Game4Loc is a large-scale cross-view UAV geo-localization benchmark using synthetic data from GTA V. It contains 120k drone-view and satellite image pairs with full 360-degree coverage.

Only final flagship method results on the official dataset protocol are listed. Automatically reviewed rows are marked `verified=false` until manual checking.
