# University-1652

University-1652 is a multi-view multi-source benchmark for drone-based cross-view geo-localization. It contains 1,652 university buildings, each captured by drone (oblique), satellite (orthogonal), and ground cameras. The standard split uses 701 buildings for training and 951 for testing. The primary task is cross-view image retrieval: given a query drone image, retrieve the corresponding satellite image (Drone→Satellite), and vice versa (Satellite→Drone). The official evaluation protocol uses Recall@K (R@1, R@5, R@10) and Average Precision (AP) computed over all queries.

Only final flagship method results on the official dataset protocol are listed. Automatically reviewed rows are marked `verified=false` until manual checking.
